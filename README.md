# Running

Just serve `dist/` and you're good to go.

# Development

Run `npm install`.

* `npm run serve` - Development server
* `npm run build` - Build
* `npm run watch` - Watch for changes (not needed with dev server)
* `npm run eslint` - Run eslint pass
* `npm run stylelint` - Run stylelint pass
